Este é um parágrafo simples em um documento HTML. A tag p é fundamental para organizar textos.

Aqui temos um texto com ênfase (em) - usado para ênfase semântica, e texto com forte importância (strong) - usado para conteúdo de maior relevância.

Exemplos de formatação: Negrito (b), Itálico (i), Texto marcado (mark), Texto menor (small)

Aqui temos um texto removido e um texto inserido (del e ins), úteis para mostrar alterações no documento.

Fórmula da água: H2O (subscrito com sub)
Metros quadrados: m2 (sobrescrito com sup)

Esta é uma citação em bloco. Blockquote é usado para citações longas e adiciona recuo ao texto.

Como diz o ditado: Uma citação curta inline usa a tag q.
