Guia Completo: Imagens em HTML e CSS

1. Imagem Básica

Explicação: Uso básico da tag img com atributos essenciais:

  • src: caminho da imagem
  • alt: texto alternativo para acessibilidade
  • width/height: dimensões da imagem

2. Imagem com Efeitos

Explicação: Imagem com efeitos CSS:

  • Borda arredondada (border-radius)
  • Sombra (box-shadow)
  • Efeito hover com escala (transform)

3. Imagem como Plano de Fundo

Explicação: Usando imagem como background:

  • background-image: define a imagem
  • background-size: cover: cobre todo o container
  • background-position: center: centraliza a imagem

4. Filtros CSS em Imagens

Explicação: Diferentes filtros CSS:

  • grayscale(): escala de cinza
  • blur(): desfoque
  • brightness(): brilho
  • contrast(): contraste
  • sepia(): efeito sépia

5. Grid de Imagens

Explicação: Grid responsivo de imagens:

  • Usa CSS Grid para layout
  • Colunas se ajustam automaticamente
  • object-fit mantém proporção

6. Elemento Picture para Responsividade

Explicação: Elemento picture para imagens responsivas:

  • Permite diferentes imagens para diferentes telas
  • Usa media queries para selecionar a imagem apropriada
  • Fornece fallback com tag img

7. Imagem com Overlay

Efeito Overlay

Explicação: Overlay com efeito hover:

  • Usa posicionamento absoluto
  • Transição suave na opacidade
  • Texto centralizado sobre a imagem

8. Figura com Legenda

Explicação: Usando figure e figcaption:

  • figure agrupa imagem e legenda
  • figcaption adiciona descrição
  • Melhora a semântica do documento

Dicas de Acessibilidade para Imagens

  • Sempre use atributo alt com descrição significativa
  • Use figure e figcaption para imagens que precisam de contexto
  • Considere usar aria-labels quando necessário
  • Mantenha bom contraste em textos sobre imagens
  • Teste o site sem imagens para garantir boa experiência

Dicas de Performance com Imagens

  • Otimize as imagens antes de usar (comprima sem perder qualidade)
  • Use formatos modernos como WebP quando possível
  • Especifique width e height para evitar layout shift
  • Use loading="lazy" para imagens abaixo da dobra
  • Considere usar srcset para diferentes resoluções
