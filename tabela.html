5. Elementos de Tabela HTML

  • <table> - Define uma tabela
  • <caption> - Define um título/legenda para a tabela
  • <thead> - Agrupa o conteúdo do cabeçalho
  • <tbody> - Agrupa o conteúdo do corpo
  • <tfoot> - Agrupa o conteúdo do rodapé
  • <tr> - Define uma linha
  • <th> - Define uma célula de cabeçalho
  • <td> - Define uma célula de dados
  • <colgroup> e <col> - Definem grupos de colunas

6. Dicas de Acessibilidade

  • Use <th> para cabeçalhos de linha e coluna
  • Use scope="col" ou scope="row" em células de cabeçalho
  • Adicione <caption> para descrever o conteúdo da tabela
  • Mantenha a estrutura da tabela simples e lógica
